Alert, Console et Document

Exercice 2

L'objet window est l'objet principal dans le modèle objet du JavaScript. Il permet d'accéder à l'ensemble des éléments du document.
  • window.alert(): Méthode qui affiche un message d'information dans une boîte de dialogue modale
  • window.console: Objet console pour afficher des informations dans la console du navigateur, en particulier avec log()
  • window.document: Objet document visible sur la page qui donne accès à tout le DOM

Exercice 1

Où écrire mon code Javascript ?
Je peux écrire du Javascript soit dans un fichier .js, soit à l'intérieur d'une balise <script> </script>.
Pour charger et exécuter le contenu d'un fichier .js, je dois renseigner le chemin du fichier dans une balise <script> via son attribut src.
Exemples: 
<script>alert('Hello world');</script>
<script src="js/exercice1.js"></script>
Une balise script peut se placer n'importe où dans une page html.
Attention cependant à toujours garder en tête qu'une page html sera lue par le navigateur de haut en bas !
